# Sena-webとは（Overview）

収集した日本国内株式の情報を出力する、  
Play Framework 2.3ベースのWebアプリケーションです。  



## ダウンロード（Download）

バイナリをダウンロードする場合は [こちら](https://github.com/yellow-man/sena-web/releases) です。
